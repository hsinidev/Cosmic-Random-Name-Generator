-   **For Developers:** Stop using risky production data. Quickly populate your development databases with realistic-looking names for robust testing.

## How to Use

1.  **Select Quantity:** Choose how many names you need (1-50).
2.  **Filter by Gender:** Select 'Male', 'Female', or 'Any'.
3.  **Choose Format:** Decide if you want a 'Full Name', 'First Name Only', or 'Last Name Only'.
4.  **Click Generate:** Hit the "Generate Names" button.
5.  **Copy Your List:** Click "Copy List" to instantly save the names to your clipboard.

-   **Core Logic:** Pure TypeScript/JavaScript (no backend required)

## Local Development

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hsinidev/cosmic-random-name-generator.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd cosmic-random-name-generator
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm start
    ```

The application will be available at `http://localhost:3000`.
